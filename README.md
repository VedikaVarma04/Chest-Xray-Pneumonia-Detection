# 🩻 AI Chest X-ray Pneumonia Detection

## 📌 Project Overview
This project uses Deep Learning to classify chest X-ray images as either **NORMAL** or **PNEUMONIA**.

Two models were implemented:
- Custom CNN
- ResNet50 (Transfer Learning)

After comparing both models, the CNN achieved higher accuracy and was selected for deployment.

---

## 📊 Model Performance

| Model | Accuracy |
|--------|----------|
| CNN | 92.14% |
| ResNet50 | 89.10% |
